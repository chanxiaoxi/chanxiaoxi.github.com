Redis安全配置

11月 15, 2015 发布在 Security

这几天网上爆出关于Redis的一个漏洞《Redis 未授权访问缺陷可轻易导致系统被黑》,这个漏洞简单的说就是利用Redis的相关方法将自己的公钥直接写入目标服务器，进而控制目标服务器。Redis现在在开发中已经被广泛使用，从缓存到数据库，但是在使用的过程中至少我很少注意Redis的安全性，从漏洞报告中可以看出，很多人都没有绑定IP，都在用默认端口，都没有设置访问密码。结合Redis官方文档关于安全配置的说明和上文中提到的攻击方式，我们来看看使用Redis时的安全配置。 阅读全文
