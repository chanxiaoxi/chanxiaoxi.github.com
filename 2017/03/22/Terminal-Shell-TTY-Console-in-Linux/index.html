Terminal Shell TTY Console in Linux

3月 22, 2017 发布在 Linux

在学习Linux的时候，我们常常会被Terminal/Shell/TTY/Console这些术语给搞混，有时候它们好像是一回事，有时候又不是，那么他们的区别到底是什么？

从字面意思来理解

  • Terminal: 电缆的末端
  • Shell: 乌龟的壳，外壳
  • TTY: Teletypewriter 电传打字机的缩写
  • Console: 柜子

Linux中的意义

  • Terminal = TTY (文本输入输出的环境)
  • Console (物理终端)
  • Shell (命令行解释器)

Terminal和TTY其实是同一个东西，在Linux中，它其实就是人机交互的窗口。比如我们常用的ssh/screen/xterm等。

Shell是Linux的应用程序，它能够提供一个运行环境（home）让用户执行命令，运行程序。常用的Shell有bash/zsh/fish等

Console是物理的终端，比如键盘、显示器等输入输出设备。
