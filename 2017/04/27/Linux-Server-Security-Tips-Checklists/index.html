一份Linux服务器安全配置清单

4月 27, 2017 发布在 Linux, Operator System, Security

这篇文章主要是收集了一些有关Linux服务器安全配置的建议和具体处理方法。

1 重要配置文件安全

1.1 用户权限配置文件权限设置

建议：修改文件的所有者和所有组均为root,修改文件的操作权限。

1.2 访问控制配置文件的权限设置

2 SSH安全

2.1 设置SSH空闲超时退出时间

建议：将超时时间设置为300-900，5-15分钟

2.2 设置SSH MaxAuthTries

建议：最大登录次数，MaxAuthTries建议值为3到6

2.3 设置SSH Log Level

2.4 设置SSHD强制使用V2安全协议

2.5 禁止SSH空密码用户登录

3 密码安全

3.1 密码复杂度检查

3.2 设置密码修改最小时间间隔

3.3 检查密码重用是否受限制

3.4 确保密码到期警告天数为7或者更多

4 账户安全

4.1 确保root是唯一的UID为0的账户

4.2 检查系统空密码账户

5 其他安全

5.1 开启地址空间布局随机化

5.2 确保rsyslog服务已启用
